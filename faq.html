常见问题 FAQ

新手问题

如何加入 Minzzj 服务器？

在游戏内输入服务器 IP：minzzj.fun:32779 或备用 IP play.simpfun.cn:32779，点击“加入服务器”即可。

服务器支持哪些版本？

目前支持 Java 1.21。请确保客户端版本一致，否则无法连接。

如何获取资源？

生存模式通过砍树、挖矿、狩猎等正常方式获取，也可与玩家交易稀有资源。

如何与其他玩家互动？

使用聊天功能交流，或加入 QQ群 62083177 实时沟通。

游戏问题

连接失败怎么办？

检查网络、确认 IP 正确；仍有问题使用备用 IP 或联系管理员。

游戏卡顿如何解决？

关闭高占用程序、降低画质；若持续卡顿请向客服反馈。

如何获取游戏币？

参与活动、完成任务或捐助服务器均可获得游戏币。

常用指令有哪些？

/tp <玩家名> 传送、/spawn 回到出生点、/msg <玩家名> <消息> 私聊。
